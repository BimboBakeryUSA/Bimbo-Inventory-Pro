# Bimbo Inventory Pro v3

Web App/PWA para conteo de inventario por ruta en warehouse.

## Nuevo en v3
- Imagen por defecto si el producto no tiene foto.
- Botón Agregar producto.
- Puedes escanear un UPC y agregar SKU, nombre, unidades por caja y URL de imagen.
- Botón Editar datos en cada card.
- La base se guarda en el navegador usando localStorage.

## GitHub Pages
Sube todos los archivos al repositorio y activa GitHub Pages en Settings > Pages.
